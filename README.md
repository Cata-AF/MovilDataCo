
# Correr proyecto:

Crear venv

    python -m venv venv

Entrar al venv

    venv/Scripts/activate

Entrara a la carpeta del proyecto

    cd MovilDataCo

Instalar dependencias

    pip install -r requirements.txt

Correr el proyecto

    py manage.py runserver

:80

    py manage.py runserver 0.0.0.0:80
